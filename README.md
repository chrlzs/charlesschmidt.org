## üöÄ Website Features

This resume website showcases modern web development practices:

### Performance Optimizations
- **Modular JavaScript**: Separated concerns with dedicated modules for animations and UI interactions
- **Lazy Loading**: Animations only initialize when visible and respect user motion preferences
- **Resource Optimization**: Preloaded critical resources and optimized asset delivery
- **Memory Management**: Proper cleanup of event listeners and animation frames

### Accessibility
- **Semantic HTML**: Proper use of ARIA labels and semantic elements
- **Motion Preferences**: Respects `prefers-reduced-motion` for users with vestibular disorders
- **High Contrast**: Support for `prefers-contrast: high`
- **Screen Reader**: Proper labeling and hidden decorative elements

### SEO & Performance
- **Meta Tags**: Comprehensive meta tags for social sharing and search engines
- **Structured Data**: JSON-LD schema markup for better search engine understanding
- **Performance**: Optimized CSS with `will-change` and `contain` properties
- **Print Styles**: Optimized print stylesheet for resume printing

### Modern CSS Features
- **CSS Custom Properties**: Consistent theming with CSS variables
- **Glass Morphism**: Modern glass effect design with backdrop filters
- **Responsive Design**: Mobile-first approach with flexible grid system
- **Dark/Light Mode**: System preference detection and appropriate styling
